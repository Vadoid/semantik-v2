# **App Name**: BigQuery Explorer

## Core Features:

- Data Connection: Securely connect to BigQuery projects using authorized credentials.
- Schema Browser: Browse BigQuery datasets, tables, and schemas with an interactive UI.
- Query Builder: Construct SQL queries via UI, or manually input them.
- AI-Powered Query Improvement: An AI tool to improve submitted query to be cost efficient and take into account all table columns.
- Data Visualization: Generate charts and graphs from query results to visualize insights.
- Insight Generation: Apply statistical analysis techniques and AI-powered insights from data exploration.
- Reporting: Create, schedule, and share generated dashboards/reports.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to convey professionalism and trust.
- Background color: Very Light Grey (#F5F5F5), providing a clean and modern look.
- Accent color: Teal (#008080), used for interactive elements and highlights, providing clarity without distraction.
- Body and headline font: 'Inter', a grotesque-style sans-serif font for a modern, machined, objective, neutral look.
- Code font: 'Source Code Pro' for displaying SQL queries.
- Simple, clear icons from a standard library like Material Design Icons.
- A clean, structured layout with clear divisions for easy navigation and data presentation.
